# React Product Grid

A responsive product gallery built with React.js and CSS.

## 🚀 Features
- **Responsive Layout:** CSS Grid with auto-fit for all screen sizes.
- **Modern UI:** Pixel-perfect cards with hover effects and wishlist integration.
- **Dynamic Content:** Data-driven components with multi-line text truncation.
